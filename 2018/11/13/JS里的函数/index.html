JS里的函数

Posted on 2018-11-13

1.函数的五种声明方式:

  • function name(参数){ return }
  • var f = function (参数）{ return } //匿名函数
  • var f = function name(参数){ return } // 具名函数
  • new Function(‘x’,’y’,’return x+y’) // 全局变量声明
  • (x,y) => { return x+y} // 箭头函数

箭头函数后面如果只用执行一句话 可以把return和{}一起省略例如：
(x,y) => { return x+y} / (x,y) => x+y;
如果箭头函数前面的参数只有一个，圆括号也可以省略，例如：
(n) => nn / n => nn ;
变量声明可以是Number String Boolean null undefined Symbol Object
而函数声明只能是 函数。
以下两种声明方式具有不一致性：

function y(){}
console.log(y) // 可以直接调用
////////////////////////////////////
var x = function y(){}
console.log (y)     //y不能被调用，调用仅限在{}内部可以调用

3.函数的本质

可以执行代码的对象就是函数，函数就是可以反复使用的代码块。函数可以接受输入的参数，不同的参数会返回不同的值。

f与f.call()的区别

f知识一个函数对象
f.call()是f的方法 表示执行这个函数的函数体
f(1,2) ========= f.call(undefined,1,2)
【this】，【arguments】
f.call(undefined,1,2)其中undefined就是this，1,2就是arguments
call的第一个参数可以用this
call后面的参数可以用arguments
