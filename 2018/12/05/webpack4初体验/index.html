webpack4初体验

Posted on 2018-12-05

webpack安装

1.首先卸载你之前配置失败的webpack

卸载webpack：
npm uninstall webpack -g

2.安装yran

在安装了nodejs的前提下安装yarn\
npm install -g yarn

3.准备安装环境

新建demo文件夹\
打开vscode或者gitbash进入目录\
初始化demo环境：npm init -y

4.下载安装

1.chrome 搜索webpack github\

2.用npm下载安装：npm install –save-dev webpack\
用yarn下载安装：yarn add webpack –dev

5.入门学习–点击进入get started


1.接下来按要求配置文件\

2.发现还有要安装的webpack-cli，那接续安装\
npm install webpack webpack-cli –save-dev或者yarn add webpack webpack-cli –dev\
核对下当前的目录结构

6.webpack的配置文件

根据Using a Configuration 提示内容新建wenbpack配置文件并复制配置内容

7.上面的配置目前只能简单转化js文件，下面我们安装scss转化css以及es6转化es5需要的配置

搜索webpack scss loader github\
按要求安装文件：yarn add sass-loader node-sass webpack –dev\
接着装： yarn add style-loader css-loader –dev\
继续装： yarn add babel-loader @babel/core @babel/preset-env webpack –dev

8.webpack.config.js文件配置内容

好了 到此为止webpack可以使用babel-loader和sass-loader分别转译scss和es6了

接下来我们应该装webpack自动化转译的配置 webpack-dev-server

yarn add webpack-dev-server –dev\
开启此功能的命令：npx webpack-dev-server\
这一步的源代码 https://github.com/coveru/first-webpack-demo
